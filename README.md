# Smart Log Analyzer

## Project Overview
Smart Log Analyzer is a Python-based project that analyzes log files to detect errors, patterns, and system insights using data analysis techniques.

## Features
- Analyze log files
- Detect errors and warning messages
- Identify patterns in logs
- Generate useful insights

## Technologies Used
- Python
- Pandas
- Jupyter Notebook

## Project Files
- `smart_log.ipynb` → Main notebook file
- `sample.log` → Sample log file for testing

## How to Run
1. Download the repository
2. Open Jupyter Notebook
3. Run `smart_log.ipynb`
4. Load log file and analyze data
